# Турникмен / Pullup Bot

Telegram-бот для программы подтягиваний с трекингом прогресса, стрика, истории тренировок и мотивацией внутри команды.

## Что умеет бот

- регистрация по секретному коду
- дневной план по волновому циклу нагрузки
- логирование тренировок (подходы, RPE, заметки)
- статистика, история и недельные итоги
- рейтинг участников и мотивация друзей
- приветствие новых участников через кнопку в чате
- базовые админ-команды и мониторинг ошибок

## Стек

- Python
- aiogram
- SQLite
- APScheduler

## Структура проекта

- `pullup_bot/` — основной код бота
- `pullup_bot/handlers/` — обработчики команд и сообщений
- `pullup_bot/services/` — бизнес-логика и фоновые задачи
- `pullup_bot/tests/` — тесты
- `privacy_policy_site/` — страница политики конфиденциальности

## Локальный запуск

Создайте файл `~/.env.pullup_bot` с переменными:

- `PULLUP_BOT_TOKEN`
- `PULLUP_SECRET`
- `ADMIN_TG_ID`
- `GROQ_API_KEY`

Далее:

```bash
python -m pullup_bot
```

## Деплой (Ubuntu)

Обычный процесс:

1. `git pull origin main`
2. `bash ~/restart-bot.sh`

Секреты должны храниться только на сервере (в `~/.env.pullup_bot`) и не коммититься в Git.
