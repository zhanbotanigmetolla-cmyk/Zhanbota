# Турникмен / Pullup Bot

Telegram-бот для систематического увеличения количества подтягиваний. Строит персональный план, адаптирует нагрузку, отслеживает прогресс и мотивирует команду.

## Возможности

### Тренировочный план
- Волновой 7-дневный цикл: Средний → Лёгкий → Тяжёлый → Отдых → Плотность → Лёгкий → Отдых
- Автоматическая прогрессия: после стабильного цикла база +5%
- Корректировка по RPE: высокий RPE → нагрузка снижается, низкий → повышается
- Учёт дополнительной активности (бег, зал) при расчёте нагрузки на следующий день

### XP и ранги (CS:GO)
18 рангов от Silver I до The Global Elite (0 – 70 000 XP):

| Группа | Ранги | XP |
|---|---|---|
| Silver | I · II · III · IV · Elite · Elite Master | 0 – 4 000 |
| Gold Nova | I · II · III · Master | 5 500 – 13 500 |
| Master Guardian | I · II · Elite · DMG | 18 000 – 36 000 |
| Eagle | Legendary Eagle · LEM | 44 000 – 53 000 |
| — | Supreme Master First Class | 63 000 |
| — | The Global Elite | 70 000 |

Начисление XP: +1 за каждое подтягивание, +50 за каждый день стрика подряд.

### Стрик и заморозки
- Стрик считает дни подряд с тренировкой или отдыхом
- Токены заморозки защищают стрик при пропуске дня
- Зарабатываются: каждые 7 дней стрика, при повышении ранга, при новом личном рекорде
- Максимум 5 токенов

### Кочка недели
Каждый понедельник автоматически выбирается участник с наибольшим числом подтягиваний за прошедшую неделю. Все участники получают уведомление с топ-3, победитель носит значок 👑 в рейтинге и статистике до следующего понедельника.

### Социальные функции
- Список участников с прогрессом каждого (сегодняшний план, стрик)
- Отправка мотивации друзьям одной кнопкой
- Уведомления о новых участниках с кнопкой приветствия
- Недельный рейтинг по подтягиваниям

### Турникмен AI
Полноценный чат-ассистент на базе **Google Gemini 3 Flash** (`gemini-3-flash-preview`, бесплатный тир).

- Знает полную историю тренировок пользователя, его ранг, стрик, базу и план на завтра
- Поддерживает многоходовой диалог — помнит контекст до 10 предыдущих обменов
- Может ответить на любой вопрос о тренировках, технике, функциях бота или личном прогрессе
- Лимиты бесплатного тира: 15 запросов/мин, 1 500 запросов/день; при превышении бот сообщает об этом пользователю
- Системный промпт содержит полную базу знаний о боте и актуальные данные пользователя — ответы персонализированы с первого сообщения

### Прочее
- История тренировок с постраничной навигацией по неделям
- Редактирование прошлых дней (в т.ч. удаление с откатом XP и program_day)
- Настройка времени уведомлений, имени, веса, нормы
- Токен заморозки при пропуске с указанием причины (без сброса стрика)
- Многоязычность: русский и английский
- Полная админ-панель: управление пользователями, рассылка, баг-репорты, статистика, режим обслуживания

## Стек

- Python 3.12
- aiogram 3.x
- aiosqlite (SQLite)
- APScheduler
- Groq API (Llama 3.3 70B)

## Структура проекта

```
pullup_bot/
  handlers/       # обработчики: training, stats, friends, settings, admin, ...
  services/       # бизнес-логика: xp, scheduler, ai
  config.py       # константы: WAVE, LEVEL_THRESHOLDS, LEVEL_NAMES
  db.py           # все операции с БД
  i18n.py         # строки на RU/EN
  keyboards.py    # клавиатуры
  states.py       # FSM-состояния
```

## Локальный запуск

Создайте `~/.env.pullup_bot`:

```
PULLUP_BOT_TOKEN=...
PULLUP_SECRET=...
ADMIN_TG_ID=...
GROQ_API_KEY=...
```

```bash
python -m pullup_bot
```

## Деплой на сервер

```bash
# из локальной папки проекта
./deploy.bat
```

`deploy.bat` пушит в GitHub, затем по SSH запускает `~/deploy.sh` на сервере (git pull → cp → systemctl restart).

Секреты хранятся только на сервере в `~/.env.pullup_bot` и не коммитятся в Git.
